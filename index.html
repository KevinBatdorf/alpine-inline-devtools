What is it?

Alpine Inline DevTools is an obverser tool that's useful when developing with Alpine.js.

To check it out, you can click on the button to the bottom right of the screen. Alternatively, while on this demo, clicking a theme to the left will also load the DevTools.

Try it out

The following string is actually an Alpine.js component. In the dev tools find the string and either press the text, or the pencil icon to update the string in real time:
